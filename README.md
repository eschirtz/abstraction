# Fruitile Survival

## Character builder
- Variables
    - Name (String)
    - Age (Number)
    - isHappy (Boolean)
- Arrays
    - Head, Body, Leg swapping (arrays)
    - Public image urls (so they can add their own info)
- Conditionals (if / else)
    - Dig get hit by fruit (passed in variables)
    - Hit someone by fruit
